# 测试数据集说明

本目录是一套面向本地课程 RAG Agent 的手工测试资料。使用时可将 `data/config.json` 中的 `root_folder` 指向本目录。

覆盖目标：

- 正常课程扫描、文件树、多级目录、文本预览和 Markdown 渲染。
- RAG 检索中的明确事实、同义问法、跨文件问题、重复概念和引用溯源。
- 文件命名边界：中文、空格、括号、大小写扩展名、很长文件名。
- 空课程、不支持扩展名、隐藏文件、损坏文档、伪装扩展名等异常资料。
- 上传落盘目录 `拖入资料`、同名文件冲突和边界文件名。

建议测试问题：

1. 数据结构课程里，栈和队列的主要区别是什么？
2. 操作系统课程里，FIFO 和 LRU 页面置换有什么区别？
3. 请根据课程资料说明死锁的四个必要条件。
4. 哪个文件提到了红黑树？
5. 这个课程有没有讲 2026 年最新考试安排？
6. 请总结“边界命名与编码”课程里的文件命名风险。

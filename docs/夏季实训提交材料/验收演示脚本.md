# 验收演示脚本

建议演示 8 至 12 分钟，所有操作使用演示资料和脱敏配置。

## 1. 开场与启动（1 分钟）

说明项目定位：本地课程资料管理、可追溯 RAG 问答和多对话学习工作台。

```bash
./start.sh
```

Windows 使用 `start.bat`。说明 Vue 前端会先构建，再由 Python 服务在 `http://127.0.0.1:8000` 提供页面和 API。

## 2. 课程、对话和资料（2 分钟）

1. 打开设置菜单，展示资料根目录和“构建知识库”。
2. 选择操作系统或数据结构课程。
3. 新建、切换并重命名一个对话，说明消息和记忆按对话隔离。
4. 展开文件树，预览 Markdown/TXT，演示当前文件搜索。

不要演示学习计划、课程概览或掌握度区块：它们不是当前前端能力。

## 3. 知识库和问答（3 分钟）

1. 构建当前课程知识库，说明解析、质量评估、切片、lexical/vector 索引。
2. 选择“答疑”，提问“页表有什么作用？”。
3. 展开“当前思考”和“处理过程”，说明 ReAct planner 可选择课程检索、联网、组合工具或澄清。
4. 点击课程引用，在右侧核对原文。
5. 切换“启发提示”或“复习”，说明模式同时影响工具与回答策略。

## 4. 学习产物和笔记（2 分钟）

1. 生成摘要和练习题，展示 `AI生成/` 文件。
2. 说明生成目录不重新进入知识库。
3. 打开课程笔记，新增或编辑一条演示笔记。
4. 切换对话，说明课程笔记仍共享。
5. 清空当前对话，确认其他对话和笔记保留。

## 5. 工程质量和边界（2 分钟）

展示 `docs/acceptance-report.md` 中实际运行的后端单测、前端测试/typecheck/build 和 RAG baseline。说明：

- 当前工作区的后端测试、前端测试、typecheck 与生产构建均已通过；外部 LLM、Web Search MCP、MinerU 和跨平台脚本仍按验收报告列为未覆盖项。

- 真实配置、消息、索引和附件不提交。
- 备份只含 `config.example.json`、`course_memory/**`、`indexes/**`。
- LLM、embedding、rerank、Web Search 和 MinerU 是外部可选服务；持续网络失败会明确报错。
- 当前系统面向单机本地使用，不应直接暴露公网。

## 6. 总结

已实现：本地课程识别和预览、多对话、三模式 ReAct、hybrid RAG、引用、流式 Markdown、摘要、练习题、笔记、配置诊断、评测和备份。限制：复杂文档/OCR、外部 provider 质量、单机文件型存储，以及未在当前前端暴露的 Dashboard/mastery API。

## 2. 课程目录扫描

来源：`local_course_agent/scanner.py`

```python
def scan(self) -> List[Dict]:
    if not self.root.exists() or not self.root.is_dir():
        raise ValueError(f"资料根目录不存在或不是文件夹: {self.root}")

    courses = []
    for course_dir in sorted(self.root.iterdir(), key=lambda p: p.name.lower()):
        if not course_dir.is_dir() or course_dir.name.startswith("."):
            continue
        children = self._scan_children(course_dir)
        courses.append({
            "id": stable_id(str(course_dir)),
            "name": course_dir.name,
            "path": str(course_dir),
            "children": children,
            "file_count": self._count_files(children),
        })
    return courses
```

一级目录是课程边界，隐藏目录被过滤；稳定 ID 同时用于 API、索引和本地状态目录。

## 3. ReAct 工具循环

来源：`local_course_agent/api/chat/flow.py`

```python
for turn in range(3):
    self.emit({"type": "status", "stage": "thinking", "detail": "正在思考下一步…"})
    step = self._plan_step(
        question,
        mode=mode,
        course_name=course.get("name", ""),
        previous_messages=previous_messages,
        has_attachments=bool(uploads),
        observations=observations,
        ai_config=ai_config,
    )
    self.emit({
        "type": "thought",
        "action": step.action,
        "detail": step.reason or react_action_label(step.action),
        "query": step.query[:120] if step.query else "",
    })
    if step.action == "final":
        planner_fallback_answer = step.answer
        break
    if step.action == "clarify":
        clarification_answer = step.answer
        needs_clarification = True
        break
```

planner 的 action 决定是否继续执行课程检索或联网；工具结果作为 observation 返回下一轮，同类工具不会重复执行。`final` 只表示 ReAct 阶段结束，之后仍由 responder 生成最终自然语言回答。课程或网页工具已经命中可用证据时会直接进入 responder，避免二次 planner 判断。模式策略同时注入 planner 和 responder。

## 5. 旧单会话迁移

来源：`local_course_agent/store/conversations.py`

```python
legacy_messages_path = self._legacy_messages_path(course_id)
legacy_memory_path = self._legacy_memory_path(course_id)
legacy_messages = (
    self._read_json(legacy_messages_path, [])
    if legacy_messages_path.exists() else []
)
legacy_memory = (
    legacy_memory_path.read_text(encoding="utf-8")
    if legacy_memory_path.exists() else ""
)
self._write_json(
    self._conversation_messages_path(course_id, "default"),
    legacy_messages,
)
self._write_text(
    self._conversation_memory_path(course_id, "default"),
    legacy_memory,
)
```

第一次建立 `conversations.json` 时，旧消息与记忆复制到 `default` 对话；原文件不删除，降低迁移和回滚风险。

## 6. 引用后处理

generation = chat_steps.postprocess_generation(answer, sources.citations)

## 7. 报告使用建议

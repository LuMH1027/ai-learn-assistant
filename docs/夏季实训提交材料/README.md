## 提交提醒

- 不提交真实 `data/config.json`、API key、token、`.venv/`、`frontend/node_modules/`、`web/dist/` 或个人课程资料。
- 封面中的班级、学号、指导教师和日期需按学校模板补全。
- 测试结论必须引用 [`../acceptance-report.md`](../acceptance-report.md)，不能把局部测试计划写成全系统通过。

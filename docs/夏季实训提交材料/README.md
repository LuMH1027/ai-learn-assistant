# 夏季实训提交材料

本目录专门汇总 PDF《夏季实训任务说明》中要求准备的材料，便于组长杨哲涵统一提交。

## 1. 大型实习报告

- `大型实习报告.md`：按九章结构整理，包含封面信息、目的要求、可行性报告、需求分析、概要设计、详细设计、程序模块、测试与用户手册、个人体会。
- `系统设计.md`：补充系统结构图、数据流图、用例图、E-R 图和关键流程图。
- `程序模块与代码摘录.md`：报告第七章可用的关键代码与模块说明。
- `使用说明.md`：报告第八章用户手册可用内容。

## 2. 过程与分工材料

- `团队分工与进度.md`：四人分工，杨哲涵为组长，陆铭皓、王子欣、徐若轩共同承担开发、测试、设计和文档工作。
- `实训日志.md`：四周实训过程记录。
- `验收演示脚本.md`：线上验收演示流程和讲解要点。

## 3. 提交与检查材料

- `提交材料清单.md`：电子代码、报告、截图和提交前检查项。
- `报告材料总览.md`：老师要求与项目材料的对应关系。
- `验收清单.md`：功能验收项和当前状态。

## 5. 提交提醒

- 提交代码时不要包含真实 `data/config.json`、API key、token、`.venv/`、`frontend/node_modules/` 等本地私有或可重建内容。
- 报告封面中的班级、学号、指导教师、日期等信息需按学校模板手动补全。
- 最终由组长杨哲涵统一在学习通提交电子代码和大型实习报告。

## 3. 贡献说明

本项目由杨哲涵担任组长，负责整体进度、任务协调、接口联调配合和最终材料提交。陆铭皓重点承担 RAG 检索、知识库构建、模型接入和系统核心逻辑；王子欣重点承担前端交互、资料预览、用户流程、测试和用户手册；徐若轩重点承担摘要练习题、学习计划、掌握度、错题订正、设计图表和报告格式。四名成员都参与了开发、测试和验收准备，分工不同但工作量保持均衡。

## 4. 组内协作方式

- 需求和验收标准先统一到文档，再落实到功能和测试。
- 开发过程中按照模块拆分问题，优先保证主流程可运行。
- 报告材料引用仓库中的真实实现和截图，避免写成空泛设计。
- 最终由组长统一提交电子代码和大型实习报告。

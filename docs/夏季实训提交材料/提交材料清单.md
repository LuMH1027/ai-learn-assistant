# 提交材料清单

根据夏季实训说明，本项目需要准备电子代码和大型实习报告，由组长统一提交。

## 1. 电子代码

提交仓库内容：

- `local_course_agent/`：Python 后端、RAG、学习服务、存储、诊断和备份。
- `frontend/`：Vue 3 + TypeScript 前端。
- `scripts/`：RAG 评测和备份脚本。
- `tests/`：后端和前端测试。
- `sample_materials/`：演示课程资料。
- `README.md`：项目说明、安装和启动方式。
- `start.bat`、`start.sh`：一键启动脚本。
- `install-deps.bat`、`install-deps.sh`：依赖安装脚本。
- `data/config.example.json`：脱敏配置示例。

不要提交：

- `data/config.json` 真实配置。
- 真实 API key、token、个人隐私资料。
- `.venv/`、`frontend/node_modules/`、`web/dist/` 等可重新生成内容。
- 个人临时聊天附件和非演示资料。

## 5. 提交前检查

- 确认报告题目、姓名、学号、班级等封面信息已按学校模板补全。
- 确认图表编号、标题层级、字体字号和参考文献格式符合模板。
- 确认程序模块章节没有整页粘贴代码。
- 确认用户手册使用真实截图。
- 确认代码压缩包中不包含真实密钥和个人隐私资料。
- 确认组长已在学习通提交电子代码和大型实习报告。

# 实训日志

## 第一周：选题与可行性

确定“基于 RAG 的 Windows 本地课程资料库与 AI 学习助手系统”题目，完成技术、经济、操作和时间可行性分析，明确课程识别、资料预览、知识库、引用问答和本地数据隔离需求。

## 第二周：基础主流程

完成一级目录课程扫描、多级文件树、PDF/图片/TXT/Markdown 预览、Python 本地 HTTP 服务和 Vue 三栏工作台。同步建立配置读取、资料目录设置和基础问答接口。

## 第三周：RAG 与学习能力

完成标题感知切片、BM25、RRF、持久向量索引、本地/外部 rerank、MMR 和引用后处理；补充附件、摘要、练习题、笔记、学习计划与 mastery 原型，以及配置健康、遥测和备份恢复。

## 第四周：收敛和验收

最终版本完成以下调整：

- 新增一门课程下的多对话，新建/切换/重命名/删除及独立消息/记忆。
- 聊天收敛为答疑、启发提示、复习三种模式，并由 ReAct planner 动态选择课程检索、联网或澄清。
- 增加实时思考、流式 Markdown、停止回答和 LLM 重试进度。
- 为减少侧栏复杂度，移除学习计划公开 UI/API 和 Dashboard/mastery 前端操作区；后端 Dashboard/mastery API 保留。
- 更新报告、演示脚本和 `docs/screenshots/`，运行完整验证并记录到 `docs/acceptance-report.md`。

## 后续方向

- 增强扫描 PDF、图片和复杂 DOCX 的 OCR/版面能力。
- 引入更稳定的真实 embedding/rerank 和 multi-vector 检索。
- 重新设计更贴近学生任务的 mastery/复习工作流后再接回前端。
- 增加带鉴权的远程部署方案和前端备份入口。

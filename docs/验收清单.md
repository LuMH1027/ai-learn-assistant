# 验收清单

本文用于项目检查、课程设计验收和用户手册截图准备。

## 一、启动验收

- 双击 `start.bat` 后服务正常启动。
- 浏览器能打开 `http://127.0.0.1:8000`。
- 页面显示标题、资料目录输入框、课程区、预览区、Agent 区。
- 页面显示 Ollama、MinerU 可选配置区。

## 二、课程识别验收

准备目录：

```text
资料根目录
├─ 操作系统
│  └─ chapter1.pdf
└─ 数据结构
   └─ stack.md
```

验收点：

- 输入资料根目录后，左侧出现“操作系统”和“数据结构”。
- 根目录下的普通文件不会被识别成课程。
- 课程内部多级子目录能保留层级。

## 三、文件预览验收

- 点击 PDF 后，中间区域显示 PDF 预览。
- 点击 PDF 后，通过独立窗口打开预览。
- 点击 MD/TXT 后，中间区域显示文本内容。
- 文件列表中不显示不支持的文件类型，如 mp4、zip。
- 将支持的文件拖到课程资料区后，文件保存到当前课程的“拖入资料”目录，并出现在文件树中。

## 四、知识库验收

- 选择课程后，“构建知识库”按钮可点击。
- 构建结束后提示入库文件数和片段数。
- 更换课程后，需要单独构建该课程知识库。

## 五、课程 Agent 验收

- 在某门课程内提问，系统返回答案。
- 答案下方显示来源文件和片段编号。
- PDF 文本可抽取时，引用能显示页码。
- 未命中资料的问题会提示“未在当前课程资料中找到可靠依据”。
- 不同课程的聊天记录互不影响。
- 不同课程的课程记忆互不影响。
- 课程笔记能保存，并且切换课程后笔记互不影响。
- 将支持的文件拖到聊天框后，AI 能读取该附件内容并在回答中引用“聊天附件/文件名”。

## 六、模式验收

- 答疑模式：直接回答。
- 启发模式：回答前带有启发式提示。
- 作业提示：明确说明只给思路，不直接完成作业。
- 复习模式：提示用户整理概念、易错点和自测题。
- 点击“生成摘要”后，右侧会话区出现课程复习摘要。
- 点击“生成练习题”后，右侧会话区出现自测题和引用来源。

## 七、可选增强验收

- `data/config.json` 中未配置 Ollama 时，系统仍可完成本地轻量问答。
- Ollama 地址或模型错误时，系统自动退回本地轻量回答，不影响主流程。
- MinerU Agent API 默认自动尝试解析 PDF。
- MinerU 网络不可用、文件过大或解析失败时，系统不会中断知识库构建，会自动降级到本地 PDF 文本抽取。

## 九、当前版本限制

- 当前实现为轻量本地原型，不依赖在线大模型。
- PDF 扫描版图片需要后续接入 MinerU/OCR 才能完整解析。
- DOCX 当前只识别文件，完整内容解析作为后续增强。
- 向量检索当前为轻量关键词评分，后续可升级 ChromaDB + embedding + rerank。

## 4. 用户手册章节建议

1. 运行环境与依赖安装。
2. 复制 `data/config.example.json` 到 `data/config.json`。
3. 配置资料根目录、OpenAI-compatible LLM/embedding、MinerU token。
4. 启动系统并打开网页。
5. 设置资料根目录。
6. 展开课程文件树。
7. 预览 PDF/文本/图片，并在 TXT/Markdown 当前文件内搜索定位。
8. 构建课程知识库。
9. 使用课程 Agent 提问。
10. 拖入附件或截图提问。
11. 生成摘要和练习题。
12. 保存、编辑、取消编辑或删除课程笔记。
13. 查看完整学习计划，编辑标题、类型和预计时长，切换状态或删除学习项。
14. 在掌握度区域新增知识点、记录答题对错和订正错题。
15. 通过“清空记忆”入口清空当前课程聊天和记忆。

## 5. 当前版本限制

- 当前仍是开发测试阶段，传入 PDF 建议小于 20 页。
- DOCX 已支持基础文本抽取；复杂版式、扫描件和图片文字仍建议转为 PDF 或配合更高质量解析工具。
- 当前检索使用本地 JSON 索引和 hybrid 策略，包含 BM25、RRF、持久向量、本地 rerank 和 MMR；未配置真实 embedding 时会使用本地 hash embedding fallback。
- OpenAI-compatible LLM/embedding、Web Search MCP 和 MinerU 属于可选外部服务，网络不可用或 key 错误时会降级或提示。

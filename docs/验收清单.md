## 2. 浏览器验收路径

1. 打开首页，检查三栏布局和设置菜单。
2. 设置演示资料目录并选择课程。
3. 新建、切换、重命名一个对话。
4. 展开文件树，预览 Markdown/TXT 并测试文件内搜索。
5. 构建知识库并观察状态。
6. 分别选择答疑、启发提示、复习模式提问。
7. 展开“当前思考”和“处理过程”，点击课程引用。
8. 生成摘要和练习题，确认 `AI生成/` 文件可预览。
9. 新增并编辑课程笔记。
10. 清空当前对话，确认其他对话和课程笔记保留。

## 4. 当前限制

- 系统面向单机本地使用，没有账号、鉴权和远程多用户隔离。
- DOCX 只做基础正文抽取；复杂排版、扫描 PDF 和图片文字依赖 MinerU/OCR 质量。
- 外部 LLM、embedding、rerank、Web Search MCP 和 MinerU 的可用性与额度不由项目保证。
- 本地 hash embedding 用于离线可运行，不等价于真实语义 embedding 的质量。
- 配置模型后的持续网络错误会返回请求错误，不保证所有失败都自动降级为本地答案。
